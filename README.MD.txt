
Run steps
Pre-requiste: Java 1.8

Command Line: Run options


STEP 1: 
Unzip files into you local directory

STEP 2: 
Open command prompt and run "java -jar TradeDataBatch.jar"
Example C:\Sasi\TradeDataBatch>java -jar TradeDataBatch.jar


On successful run the batch job file is created in the target folder.
Example C:\Sasi\TradeDataBatch\target\summaryReport.csv
